**Quick IPTV 8K** is a sleek, ultra-responsive media player built with React and Electron. Designed for users who demand a premium streaming experience, it gracefully scales from small laptop screens all the way up to stunning 8K Mac displays.

---

## ✨ Features

- **📺 Broad Compatibility:** Seamlessly parse and stream from **M3U Playlists**, **Xtream Codes** APIs, and **Stalker Portals**.
- **🎬 Cinematic UI:** A beautiful, distraction-free dark UI designed with high-quality aesthetics and smooth micro-animations.
- **⚡ Advanced Playback:** Powered by `hls.js` for robust handling of high-bitrate live streams (HLS/M3U8) and Video-on-Demand (VOD) formats.
- **🖥️ Ultra-Res Scaling:** A native Flexbox architecture that intrinsically adapts to any monitor size (HD to 8K).
- **🍏 Native macOS Touch:** Enjoy integrated native traffic-light window controls for a seamlessly native Mac experience.
- **👥 Multi-Profile:** Save and manage multiple provider credentials or playlists simultaneously.
- **🔍 Smart Search & Categorization:** Find live channels and VODs instantly with built-in search and grouping.

---

## 🛠️ Technology Stack

- **Framework:** [React 19](https://react.dev/)
- **Desktop Wrapper:** [Electron](https://www.electronjs.org/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Video Engine:** Native HTML5 `<video>` & [`hls.js`](https://github.com/video-dev/hls.js/)
- **Styling:** Custom Vanilla CSS for maximum performance and strict premium aesthetics.
